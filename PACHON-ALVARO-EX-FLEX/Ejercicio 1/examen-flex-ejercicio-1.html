He utilizado display: flex en la caja del ejercicio 4 para que sea flex, en esta misma caja he utilizado align-items:center para alinear las cajas(hijas) al centro del eje transversal.

Para dar tamaño a las cajas que aparecen utilize flex-grow ya que establece el valor de crecimiento de cada una para que ocupen todo el espacio disponible, cuanto mayor sea el numero del valor de flex-grow mayor sera el espacio que ocupa esta caja, pero si unas cajas tienen el mismo valor se repartiran a partes iguales el espacio sobrante para ocupar todo el espacio disponible.

Para la caja 1 utilize flex-grow:1;

Para la caja 2 utilize flex-grow:0;

Para la caja 3 utilize flex-grow:2;

Por lo cual con flex-grow etsamos calculando para que entre todas ocupen todo el espacio posible dando como resultado que la caja 1 al tener flex-grow:1; de mas grande que la 2 ya que esta tiene flex-grow:0; y a la vez que la caja 3 sea mas grande que las anteriores ya que tiene flex-grow:2; pala la caja 3 también utiliza align-self:flex-end; para situar esta caja al final del eje transversal.
